# SEO Plan for Jacek Janczura’s Personal Site

## 1. Keyword Strategy

- Name/Title: Jacek Janczura, AI Architect, CTO, Prompt Engineer, System Designer, Security.
- Expertise: Enterprise AI, Generative AI, RAG, Cloud Architect, ISO 27001, SOC 2.
- Projects/Tech: entAIngine, AI agents, KnowledgeX, confidential computing, LLM.
- Location/Context: Berlin AI CTO, AI startup co-founder Germany.
- Publications: IEEE paper, KnowledgeX whitepaper, Medium NFTs article.

Use naturally in titles, headings, and copy; localize equivalents for PL/DE.

## 2. On-Page SEO

- Titles: “Page | Jacek Janczura”, <=60 chars where possible.
- Descriptions: 140–155 chars, unique per page.
- Headings: one H1; meaningful H2s per section.
- Image SEO: descriptive filenames + alt; width/height set to avoid CLS.
- Internal links: contextual links between Home ↔ Projects ↔ Talks ↔ About.
- Schema: JSON-LD `Person` on Home; `Article` for notes.
- Hreflang: en, pl, de alternates per page.
- Sitemap + robots.txt.

## 3. Performance

- Minimal JS islands; defer non-critical.
- Astro image optimization; lazy iframes.
- Preload Lato or self-host WOFF2; `display=swap`.

## 4. Off-Page

- Add site to LinkedIn, GitHub profiles.
- Submit to Search Console/Bing Webmaster; monitor coverage.

## 5. Monitoring

- Core Web Vitals via Lighthouse.
- Check Rich Results for JSON-LD validity.
- Review queries/coverage monthly in GSC.
