# Assets & Resources Checklist

## Must-have

- **Headshot** (1000×1000, neutral bg, JPG/WEBP).
- **OG image** (1200×630, name+title+portrait).
- **Icons** (SVG): LinkedIn, Email, GitHub, Sun/Moon, Hamburger.

## Nice-to-have

- **Project visuals** (1200×800): screenshots or neutral illustrations.
- **Talk thumbnails** (optional; YouTube provides default).

## Font

- **Lato** 400/700 via Google Fonts (`display=swap`) or self-host WOFF2.

## File placement

- `public/images/` — headshot, OG, project images.
- `public/icons/` — SVG icons.
- `public/CNAME` — `janczura.com` (when ready).
